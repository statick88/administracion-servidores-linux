# Code of Conduct

Este proyecto adopta el Contributor Covenant.

## Nuestro compromiso

Nos comprometemos a mantener un espacio respetuoso y colaborativo.

## Comportamiento esperado

- Respeto y comunicacion profesional.
- Critica tecnica enfocada en el contenido.
- No acoso, no discriminacion.

## Reportes

Si necesitas reportar una conducta inapropiada, abre un issue o contacta al mantenedor.
