Administración de Servidores Linux

Abacom
Hardening · Docker · Nginx + SSL · Docker Compose

Bienvenida

  • ¡Hola a todos! Es un gusto darles la bienvenida.
  • En este curso vas a instalar y operar un servidor Linux real (o VM) con enfoque profesional.
  • Priorizamos seguridad, orden y diagnóstico: no solo “que funcione”, sino que sea operable.
  • Todo se valida con evidencias: comandos, estado de servicios, puertos y logs.

Para quién es

  • Estudiantes de TI / admins junior / devs que necesitan entender servidores.
  • Requisito: comodidad básica usando terminal (copiar/pegar comandos y leer salida).

Qué lograremos juntos

  • Hardening y seguridad: Firewall (UFW) y Fail2Ban.
  • Contenerización: instalación y operación de Docker.
  • Infraestructura web: despliegue de Nginx con SSL (Certbot).
  • Orquestación: Docker Compose (servicios + MySQL/Postgres).
  • Operación diaria: SSH, permisos, paquetes, logs y diagnóstico.

Metodología (aprender haciendo)

Concepto -> Demo -> Laboratorio guiado -> Práctica -> Reto

Cómo trabajamos (reglas simples)

  • Todo cambio se prueba con un comando verificable (estado/puerto/log).
  • Cada práctica termina con evidencias (capturas/salidas) y un checklist.
  • Si algo falla: registras el error, hipótesis y pasos para reproducir.

Entorno de trabajo (recomendado)

  • Ubuntu Server LTS (VM o equipo dedicado)
  • Acceso por SSH desde tu máquina (Linux/macOS/Windows)
  • Un editor (VS Code recomendado) y Git instalado
